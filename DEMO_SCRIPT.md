# DEMO_SCRIPT (60 seconds)

## Goal
Show Calendar Copilot turning Gmail scheduling language into an approval-first Google Calendar workflow.

## Script

1. **Open dashboard**
   - "This is Calendar Copilot desktop. It watches Gmail and proposes events, but never creates anything without approval."

2. **Show setup checklist**
   - Point at the four checks: Google connected, Gmail access, Calendar access, watcher enabled.
   - Mention that setup is explicit and visible.

3. **Show watcher status**
   - Toggle watcher on.
   - Click **Run now**.
   - Note updated `Last checked` and detected counters.

4. **Show detected event**
   - Open a candidate card.
   - Highlight title, date/time, sender, Gmail source badge, confidence, message snippet.

5. **Show approval control**
   - Click **Edit** and tweak title/time/location quickly.
   - Click **Create Event**.
   - Mention duplicate prevention (source message ID + title/start similarity).

6. **Show created events panel**
   - Confirm created item appears in **Created Events**.
   - Close with: "One click to approve, no silent calendar writes."
